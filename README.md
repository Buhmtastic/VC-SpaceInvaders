# VC-SpaceInvaders

## English

This is a Space Invaders-style game developed using Python and PyGame. This project was created with active utilization of Vive Coding principles and tools, demonstrating an iterative and collaborative development process.

## Korean (한국어)

이 프로젝트는 Python과 PyGame을 활용하여 개발된 Space Invaders 스타일의 게임입니다. Vive Coding의 원칙과 도구를 적극적으로 활용하여 반복적이고 협력적인 개발 과정을 통해 만들어졌습니다.
