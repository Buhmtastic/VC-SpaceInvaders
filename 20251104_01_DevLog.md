# 2025년 11월 4일 개발 일지

## 목표
- `reference` 폴더의 `invader.py`와 `strip.png`를 활용하여 게임의 시각적 요소 개선

## 현재 상태 분석
- `main.py`의 현재 게임 요소(플레이어, 적 등)는 모두 단색 사각형으로 구현되어 있음.
- `reference/invader.py`는 `reference/strip.png` 스프라이트 시트를 사용하여 캐릭터를 화면에 그리는 로직을 포함하고 있음.

## 1단계: 적(Alien) 이미지 변경
- **계획:** `Alien` 클래스를 수정하여, 단색 사각형 대신 `strip.png`의 외계인 이미지를 사용하도록 변경.
- **결과:** `Alien` 클래스를 성공적으로 수정하여 `strip.png`의 스프라이트를 사용하도록 변경함. 외계인이 애니메이션이 적용된 이미지로 표시되는 것을 확인함.

## 2단계: 플레이어(Player) 이미지 변경
- **계획:** `Player` 클래스를 수정하여, 단색 사각형 대신 `strip.png`의 플레이어 이미지를 사용하도록 변경.
- **결과:** `Player` 클래스를 성공적으로 수정하여 `strip.png`의 스프라이트를 사용하도록 변경함. `AttributeError`를 해결하고 플레이어가 이미지로 표시되는 것을 확인함.

## 3단계: 총알(Bullet) 이미지 변경
- **계획:** `Bullet` 및 `AlienBullet` 클래스를 수정하여, 단색 사각형 대신 `strip.png`의 '빔'과 '폭탄' 이미지를 사용하도록 변경.
- **결과:** 모든 총알이 이미지로 표시되도록 성공적으로 수정함.

## 4단계: UFO 이미지 변경 시도 및 버그 발생
- **계획:** `UFO` 클래스를 수정하여 `strip.png`의 이미지를 사용하도록 변경 시도.
- **문제:** `strip.png`에 UFO 이미지가 없어 UFO가 게임에 나타나지 않는 버그 발생.

## 5단계: 외계인 속도 조정
- **계획:** 적 유닛의 움직이는 속도가 너무 빠른 것 같아 `main.py`의 `_update_aliens` 메서드에서 속도 증가폭을 줄임.
- **결과:** `speed_multiplier`와 `wave_speed_bonus` 값을 조정하여 외계인의 움직임이 이전보다 덜 공격적으로 변경된 것을 확인함.

## 6단계: UFO 버그 수정
- **계획:** UFO가 나타나지 않는 문제를 해결하기 위해 `UFO` 클래스를 이미지 사용 이전의 코드로 되돌림.
- **결과:** `UFO` 클래스와 `_spawn_ufo` 메서드를 원래 코드로 되돌려, UFO가 다시 단색 사각형으로 정상 표시되는 것을 확인함.
